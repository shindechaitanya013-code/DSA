# Data Structures and Algorithms (DSA)

This repository contains basic to intermediate Data Structures and Algorithms programs for practice and learning.

## Topics Covered

* Arrays
* Searching (Linear Search, Binary Search)
* Sorting (Bubble, Selection, Insertion, etc.)
* Linked List (Singly, Doubly)
* Stack
* Queue (Simple, Circular)
* Recursion
* Trees (Binary Tree, BST)
* Graphs (Basic Traversal)
* Hashing
* Heap
* Greedy Algorithms
* Dynamic Programming (Basic)

## Purpose

Created for daily coding practice and strengthening problem-solving skills in Data Structures and Algorithms.
